User Guide

📜
"Lesson one from this experience: Realize how much you don't know. Lesson two: Make sure to learn from lesson one." - Jacen Solo

The following steps need to be taken to fold a protein sequence with AlphaFold2 using ColabFold:

* Upload your protein sequence as a fasta file at Upload
- The file must be a .fasta file
- Beside letters and numbers '-', '.' and '_' are the only allowed characters in the file name - no spaces
- Multimers need ':' as chain breaks like MASDFGH:KLERWQ
- The maximum length of the file name (excluding .fasta) is 15 characters
- Please submit only one protein at a time
- Example fasta files can be download at Example
* Please enter a user name you will remember
- Without your user name you won't be able to retrieve your results
- Beside letters and numbers '-', '.' and '_' are the only allowed characters in the user name - no spaces
* Enter a valid token
- One token can queue a maximum of three jobs
* Select the number of structures you want to be calculated
* Choose whether to relax your structures with amber
* Choose the number of recycles
- Usually auto determines the needed number well but for artificial proteins a higher set number might help
* After submission your job will be queued and started when the resources are available
* When your sequence is folded you can find it with the same name as your fasta file in Download

Download your results when they are finished - they will be deleted without consideration after one week

Predictions running longer than 3 days will be terminated

The following steps need to be taken to fold a protein sequence with AlphaFold3:

* Upload your protein sequence as a json file at Upload
- The file must be a .json file
- Example json files can be download at Example
- For a complete guide to build your desired input json file visit AlphaFold3 GitHub
* Enter a user name
* Enter a valid token
* Ignore all other input fields - they are meant for Alphafold2
* Otherwise same rules apply as for Alphafold2
